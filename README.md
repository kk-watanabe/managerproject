# README

## 開発管理統合アプリケーション（PoC）
企業実習で出された課題に取り組んで作成した社内開発管理アプリです。実習は作業時間が最大100時間という制約の中で、要件の読み込み、設計からテストまで一貫して行いました。<br>

<br>

開発管理に関するプロセスとデータを一元管理するアプリケーションです。従来の管理方式の課題として、申請システム、各部門での進捗管理（Excel）、本部への共有、予算管理がそれぞれ分断され非効率だったことがありました。そこで、それらを「一元管理」することができれば、管理の効率化と可視化を実現できると考えました。本アプリはその仮説に対するPoCです。<br>
一つのアプリ上で案件申請、承認フロー、案件の進捗管理、予算実績管理をすべて管理することができます。

## 主な機能一覧
【申請者の機能】
- 新規案件の申請
- 自身の申請案件の進捗入力・更新
- 予算実績の入力
- 自身の案件一覧・ステータス確認

【部門管理者の機能】
- 申請の一次承認
- 自部門の案件一覧・進捗確認

【本部管理者の機能】
- 関連部門全案件の一覧・進捗確認
- 関連部門予算消費状況の確認

【その他の機能】
- 通知機能（予算超過、承認フローの審査結果）
- コメント機能
